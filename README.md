# 🧬 RNA-Seq Gene Expression Dashboard

## 📌 Project Overview
This project demonstrates an end-to-end RNA-Seq gene expression analysis pipeline. It includes data preprocessing using Python and visualization through a Power BI dashboard.

The goal is to transform raw gene expression data into meaningful insights using data analysis and interactive visualization.

---

## 🚀 Features
- Data preprocessing using Python (Pandas)
- Gene expression transformation
- Visualization using Matplotlib & Seaborn
- Interactive dashboard creation in Power BI
- Exported plots and dashboard snapshots

---

## 🛠️ Tech Stack
- Python
- Pandas
- Matplotlib
- Seaborn
- Power BI
- VS Code (Ubuntu)
